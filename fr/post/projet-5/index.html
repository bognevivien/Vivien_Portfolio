Face mask detector

Classification: Utiliser open Cv , keras and tensorflow pour créer un détecteur du port du masque.

PROJETS

Face mask detector

February 10, 2021

Problématique:

Nous mettons sur pieds dans ce projet un détecteur de masque. Les différentes étapes sont les suivantes:

  • Utiliser OpenCV pour charger les images et les labeliser
  • Utiliser un algorithme de DL pour creer un classificateur d’Images
  • Utiliser openCV et le classificateur pour détecter le port du masque.

Nous allons utiliser comme démarche data science: CRISP-DM

  • Comprehension des données: Nous allons collecter ou télécharger les images nécessaires à l’entrainement du modèles.
  • Préparation des données: Préparer les données pour leur utilisation dans les algorithmes de modélisation
  • Modélisation et evaluation : Création d’un classificateur en créant un model ou en configurant un modèle déjà créer ( Transfer learning).
  • Utilisation et résultats : Utiliser le modèle pour détecter le masque dans des images ou vidéos.

Comprehension des données :

Il existe plusieurs moyens d’obtenir les données:

  • Soit on construit la base de données en utilisant notre caméra et openCV video capture
  • Soit on télecharge un jeu de données déjà disponible et créer par les pairs.

Nous allons effectuer les deux dans l’optique d’apprendre à manipuler openCV.

Le résultat du script çi dessous est deux dossiers : “with mask” et “whithout mask” comprenant des images avec et sans masques.

Nous allons plutôt utiliser un jeu de données plus variée collecté par les pairs et disponible dans les liens :

  • with mask: https://github.com/NVlabs/ffhq-dataset
  • without mask: https://github.com/cabani/MaskedFace-Net

Préparation des données :

Vu que les labels sont déjà séparement disponible, il s’agit de préparer les données pour leur entrée dans le réseau de neurones. Nous allons:

  • Redimensionner les données en (None, 224, 224) puisque ce sont les dimensions de la couche en entrée du modèle que nous allons utiliser.
  • Recuperer la face dans l’image à l’aide du haarcascade frontalface
  • Normaliser les données en vue d’uniformiser l’apport de chaque variable indépendante du modèle.

Modélisation et Evaluation

Il s’agit d’un transfer learning. Nous allons utiliser un modèle de classification présent dans keras: mobilenet et le configurer à notre guise en vue d’obtenir notre classificateur.
